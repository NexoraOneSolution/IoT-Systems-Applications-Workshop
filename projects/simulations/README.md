# Simulation Projects

This folder contains the beginner-friendly simulation exercises used during the workshop.

| Project | Focus | Hardware | Folder |
| --- | --- | --- | --- |
| Smart Street Light | Light-based automation | Arduino Uno, LDR, transistor, lamp | [smart-street-light](./smart-street-light/) |
| Weather Station | Multi-sensor monitoring | Arduino Uno, TMP36, gas sensor, potentiometer, LCD | [weather-station](./weather-station/) |

## How To Use These Projects

1. Open the project folder.
2. Read the local `README.md`.
3. Review the `media/` images and `bom.csv`.
4. Open the `src/` sketch in Arduino IDE or Tinkercad text mode.
5. Simulate or upload the circuit.
