## Recommended Workshop Flow

1. Start with the [getting started guide](../../docs/getting-started.md).
2. Open one project README and scan the learning outcomes.
3. Recreate the circuit using `media/` images and the `pinout.md` table.
4. Review `bom.csv` before class or before a hardware build.
5. Open the sketch inside the project `src/` folder and simulate or upload it.
6. Use the troubleshooting guide if the reading or output does not match the expected behavior.
