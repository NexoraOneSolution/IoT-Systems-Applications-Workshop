# Gas Leak Detector

## Build Goal

Monitor gas concentration and trigger a buzzer and warning LED when the reading crosses a safety threshold.

## Suggested Components

- Arduino Uno
- MQ-series gas sensor
- buzzer
- red and green LEDs
- optional LCD

## Skills Practiced

- safety alert logic
- analog sensor monitoring
- multi-output status indicators
- threshold tuning

## Stretch Ideas

- show live values on an LCD
- add a mute button
- log alert events over Serial
