# Extension Projects

These project briefs turn the workshop repo into a better follow-up lab and capstone resource. They are intentionally lighter than the two guided simulation builds and are meant to help students design the next step themselves.

| Project | Difficulty | Outcome |
| --- | --- | --- |
| [Smart Irrigation System](./smart-irrigation-system/README.md) | Beginner to intermediate | Automates watering from a soil reading |
| [Gas Leak Detector](./gas-leak-detector/README.md) | Beginner to intermediate | Triggers alerts when gas levels rise |
| [Home Security Alarm](./home-security-alarm/README.md) | Beginner to intermediate | Detects motion or intrusion and sounds an alarm |
| [Temperature Logger](./temperature-logger/README.md) | Beginner to intermediate | Records readings for later analysis |
| [MQTT Cloud Monitoring](./mqtt-cloud-monitoring/README.md) | Intermediate | Publishes sensor data to cloud dashboards |
