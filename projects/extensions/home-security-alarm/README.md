# Home Security Alarm

## Build Goal

Detect motion or door opening and trigger an alarm only when the system is armed.

## Suggested Components

- Arduino Uno
- PIR sensor or reed switch
- buzzer
- status LED
- keypad or pushbutton for arming

## Skills Practiced

- digital sensor reading
- state management
- alarm timing
- user interaction

## Stretch Ideas

- exit delay timer
- silent mode
- event counter on an LCD
