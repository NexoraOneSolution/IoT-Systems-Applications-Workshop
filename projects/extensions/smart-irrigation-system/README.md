# Smart Irrigation System

## Build Goal

Create a watering system that turns a pump or relay on only when soil moisture drops below a threshold.

## Suggested Components

- Arduino Uno or ESP32
- soil moisture sensor
- relay module or pump driver
- water pump or indicator LED
- optional LCD or OLED display

## Skills Practiced

- threshold-based control
- relay safety basics
- environmental sensing
- automation timing

## Stretch Ideas

- manual override button
- water tank level sensor
- cloud notifications for dry soil
