# Temperature Logger

## Build Goal

Capture temperature readings at regular intervals and export them in a format that can be graphed later.

## Suggested Components

- Arduino Uno
- TMP36 or DHT sensor
- RTC module or timed loop
- Serial Monitor or SD card module

## Skills Practiced

- repeated timed sampling
- structured data output
- basic data analysis thinking
- sensor calibration

## Stretch Ideas

- print CSV lines for spreadsheet import
- compare day and night trends
- add a rolling average
