# MQTT Cloud Monitoring

## Build Goal

Send sensor data from a Wi-Fi enabled board to an MQTT broker or dashboard for remote monitoring.

## Suggested Components

- ESP8266 or ESP32
- temperature, humidity, or air-quality sensor
- MQTT broker such as HiveMQ or Mosquitto
- dashboard platform such as Node-RED or Blynk

## Skills Practiced

- wireless connectivity
- publish and subscribe basics
- dashboard design
- alerting and remote visibility

## Stretch Ideas

- control an LED remotely
- push alerts when readings are out of range
- store history in a database or dashboard tool
