# Final Project Ideas

These ideas extend the workshop into mini-capstone work.

## Suggested Tracks

### Smart Irrigation System

- read soil moisture and trigger watering only when needed
- add a pump or relay stage
- send a warning when the water tank is empty

### Gas Leak Detector

- monitor air quality or gas concentration
- trigger LEDs and a buzzer when readings cross a safety threshold
- log alert events to the Serial Monitor

### Home Security Alarm

- detect movement or door opening
- trigger a buzzer and indicator light
- add armed and disarmed states

### Temperature Logger

- record temperature readings at fixed intervals
- print readings in CSV format for later analysis
- compare conditions across different times of day

### MQTT Cloud Monitoring

- publish sensor data to a dashboard
- use ESP8266 or ESP32 hardware
- add remote monitoring and alerts
