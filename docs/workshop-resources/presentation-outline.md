# Presentation Outline

Use this as a starter script for slides or a live whiteboard session.

## Slide Flow

1. Welcome and workshop goal
2. What IoT means in real products
3. Arduino Uno basics
4. Inputs, outputs, sensors, and actuators
5. Reading analog values with `analogRead()`
6. Controlling outputs with `digitalWrite()`
7. Project 1: smart street light logic
8. Project 2: weather station architecture
9. Simulation workflow in Tinkercad
10. Common wiring and debugging mistakes
11. Extension projects and real-world applications
12. Quiz, assignment, and wrap-up

## Talking Points

- Keep the first project fast so students gain momentum early.
- Use the Serial Monitor whenever students feel the circuit is "not working."
- Encourage students to predict sensor behavior before running the simulation.
- Link every workshop exercise to a real-world automation or monitoring use case.
