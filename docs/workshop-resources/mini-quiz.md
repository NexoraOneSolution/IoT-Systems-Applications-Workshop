# Mini Quiz

Use this after the guided build or at the end of the workshop.

## Questions

1. What is the difference between an input device and an output device in an IoT circuit?
2. Which Arduino function is commonly used to read an analog sensor?
3. Why is an LDR useful in the smart street light project?
4. What happens when the weather station buttons are pressed?
5. Why do many circuits require a common `GND` connection?
6. What does a resistor do in a simple Arduino circuit?
7. Why is the Serial Monitor useful during debugging?
8. What is the role of a threshold in an automation project?
9. Which project in this repo uses an LCD?
10. Name one real-world use case for an IoT monitoring system.

## Answer Key

1. An input device sends information to the controller, while an output device shows or performs an action.
2. `analogRead()`
3. It measures light intensity so the controller can decide when to turn the lamp on or off.
4. They switch the display mode between temperature, air quality, and humidity views.
5. Without a shared ground, the circuit does not have a stable reference point.
6. It limits or shapes current and voltage in the circuit.
7. It shows sensor values and state changes so logic problems are easier to diagnose.
8. A threshold decides when the code should switch from one behavior to another.
9. Weather Station
10. Any reasonable example such as weather monitoring, irrigation, home security, or air-quality alerts
