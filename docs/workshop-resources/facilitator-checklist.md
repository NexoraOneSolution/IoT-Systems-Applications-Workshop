# Facilitator Checklist

Use this before students enter the lab.

## Room Setup

- projector or large display working
- Wi-Fi or hotspot available for simulation access
- enough USB ports or extension boards available
- every seat has a breadboard and jumper wires

## Instructor Setup

- open the schedule PDF
- open the two core project READMEs
- test one simulation before the session starts
- confirm Arduino IDE and libraries are installed on demo machines

## Student Support Prep

- print or share the assignment sheet
- keep the mini quiz ready for the last session block
- prepare a fallback demo if internet access is slow
- decide which extension project you want to pitch at the end
