# Assignment Sheet

## Objective

Build one of the core workshop projects, document the wiring clearly, and explain how the logic works.

## Student Tasks

1. Choose either the smart street light or weather station project.
2. Recreate the circuit in Tinkercad or on a physical Arduino Uno.
3. Run the provided sketch and confirm the expected output.
4. Change at least one behavior in the code.
5. Record what changed and why.

## Submission Checklist

- circuit screenshot or photo
- updated sketch file
- short explanation of the logic
- one challenge faced during debugging
- one idea for improving the project

## Optional Stretch Goals

- add a buzzer or LED indicator
- adjust thresholds for your own environment
- add one more display message or serial output
- combine ideas from both workshop projects
